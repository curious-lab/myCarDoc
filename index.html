Introduction

Curious Cars - Simple Car Rental Website

 
Curious Cars - A Bootstrap 4 based basic Car Rental HTML5 Template
 
Curious Cars is fully Responsive Template. It is Multi-Device supported & Cross Browser Compatible. Well Documented. This Documentation provides you with a clear guideline for how to use and customize the template.

Features

  • Perfect Bootstrap 4 Elements.
  • Well Commented Code that helps you to easy customization.
  • Clean and W3C validate the code.
  • All necessary Elements Included etc.
  • Google Fonts.
  • Font Awesome 5 Icons.
  • Fully Responsive and Supported Multiple Modern Device like Mobile, Tablet, Desktop, etc.
  • All Modern Browser Supported.
  • Modular CSS Block Styles & W3C Validated Code.
  • Well Documented.

Javascript ↦ Ajax

This template has following javascript plugins:

  • Bootstrap js
  • jQuery js
  • jQuery Migrate js
  • Gijgo js
  • Owl Carousel js
  • Magnific Popup
  • Isotope Js
  • Vegas js
  • Slicknav js
  • popper js
  • main js

all these files are located in js folder, And our custom main.js is located in assets/ js folder in the root directory.

This template has following custom javascript functions

  • mc_week_days_def js - which has definations of week days
  • sample_cars js - which has function to get all the dummy cars
  • sample_location js - which has function to get all the dummy locations
  • cars js - which has various functions related to cars.html page
  • home js - which has various functions related to home.html page

all these files are located in js/business/data and query folder,
